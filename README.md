Un generador de contraseñas **seguras y aleatorias** hecho en **Python** 🐍.  

## 🚀 Funcionalidades
- Longitud configurable (por defecto 16).
- Incluir/excluir mayúsculas, números y símbolos.
- Uso directo desde la terminal.
